# Babylon.js 3D Model Viewer

This project is a simple 3D model viewer built using Babylon.js. It allows you to load and display 3D models in a web browser.

## Project Structure

```
babylonjs-3d-model-viewer
├── src
│   ├── index.html       # Main HTML document
│   └── main.js          # JavaScript code for Babylon.js
├── package.json         # npm configuration file
└── README.md            # Project documentation
```

## Getting Started

To run this project, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/babylonjs-3d-model-viewer.git
   cd babylonjs-3d-model-viewer
   ```

2. **Install dependencies:**
   If you have npm installed, run:
   ```bash
   npm install
   ```

3. **Open the project:**
   Open `src/index.html` in your web browser to view the 3D model.

## Features

- Displays 3D models using Babylon.js.
- Interactive controls for navigating the 3D scene.

## Additional Information

- Ensure you have a compatible web browser that supports WebGL.
- You can modify the `main.js` file to load different 3D models by changing the model URL.
